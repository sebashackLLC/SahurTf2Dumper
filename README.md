# Tf2 AutoDumper

will dump tf2 offsets - you can use ai to like add more offsets i dont think it dumps all of them 

### Setup

1. download the zip
2. unzip
3. open cmd in that folder 🧠
4. open tf2
5. run py gui.py in cmd
5. offsets.
